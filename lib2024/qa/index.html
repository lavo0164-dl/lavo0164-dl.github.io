  2. Qus: Define version control and give one example.

    Ans: Management of changes to documents with revisions timestamped and followded by the name of the contributor. Sublime Text provides a version control.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The title tag is used to create a paragraph in html. <title>

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: The p tag is used to create a paragraph in html. <p>

  5. Qus: Identify the odd one from these tags <span> <p> <a> <i> and state the reason?

    Ans: The <i> tag is the odd one, as it deals with modifying text to present as italics vs the structure and functionality aspects common to the other tags.

  6. Qus: What are the tags required to create a navigation list in HTML

    Ans: The tags required to create a navigation list in HTML are the nav tag <nav> in addition to the list tags of <ol> <ul> and <li>

  7. Qus: What element is used to create images? Name its attributes.

    Ans: The <img> is an element used to create images.

  8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    Ans: The <b> tag is the odd tag because it speaks to making text bold, whereas the other tags are applied directly in the construction of a table

  9. Qus: Give an example of color picker input.

    Ans: The value attribute sets the value of <input> elements. The name attribute specifies a name for the information/value of the content attribute for a <meta> element.The name attribute is used together with the value attribute to specify parameters for the specified with <object> tag.

  12. Qus: How would you link an external style sheet? Give example.

    Ans: To link to an external style sheet, we name the sheet styles.css and then add <link rel="stylesheet" href="css/styles.css"> in the head tag.

  13. Qus: Create CSS block to center an element both horizontally and vertically that has position:absolute and has width:200px and height:200px.

    Ans: { margin: 0; width: 200px; height: 200px; position: absolute; Top: 50%; Left: 50%; }

  14. Qus: Create CSS keyframes block called "spin" that rotates an element.

    Ans: @keyframes spin{ from{ transform: rotate(0deg); } to { transform: rotate(360deg); } }
